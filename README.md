# Ping Pong Game 🏓

A Python-based Ping Pong game with music, sound effects, custom font, and animated background.

---

## Requirements
- Python 3.x
- `pygame` library

Install pygame by running:

pip install pygame

---

## How to Run

1. Click the green **"Code"** button → **"Download ZIP"**
2. Find the ZIP in your Downloads folder
3. **Right-click ZIP** → **Properties** → check **"Unblock"** → click **OK**
4. Extract the ZIP
5. Open the extracted folder
6. Run the game:
python pingpong.py


---

## Files

| File | Purpose |
|------|---------|
| `pingpong.py` | Main game file |
| `pingpong_config.json` | Game configuration settings |
| `Boing.wav` | Ball hit sound effect |
| `miss.wav` | Sound effect when a point is missed |
| `win_music.wav` | Music that plays when someone wins |
| `game_music.wav` | Music that plays when you start the game|
| `lobby_music.wav` | Music that plays when you are in the Lobby |
| `8 Bit Cat GIF.gif` | Default background animation |
| `PressStart2P-Regular.ttf` | Pixel/retro font used in the game UI |
| `OFL.txt` | Font license file |

---

## Custom Background 🎨
Don't like the default background? You can use your own!

- Supported formats: `.gif`, `.png`, `.jpg`
- Replace/Add the background file in the game folder
